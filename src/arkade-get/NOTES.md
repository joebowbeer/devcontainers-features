Specify either a version/tag, or "latest" to get the default version.

```json
    "features": {
        "ghcr.io/joebowbeer/devcontainers-features/arkade-get:1": {
            "faas-cli": "0.14.10",
			"helm": "v3.19.0",
            "kustomize": "latest"
        }
    }
```

The binaries are placed in `/usr/local/bin/`
